Project Title: Campus Management System 

Project Description:This Campus Management System is a console-based project developed in Dev C++ that provides separate portals for Admin and Students with secure login and registration. It allows management of students, courses, and teachers, along with features like campus map navigation, route finding, timetable viewing, fresher tips, and FAQs. All records are stored permanently using file handling, making the system efficient for basic campus operations.

Tools/Technologies Used: Dev C++ and internet.

Instructions to Run the project: Campus Management System comprises of an Admin portal and a  Student portal. Admin portal allows the user add and manage courses , assign courses to teachers. On the other hand the Student portal provides students with Campus map , Campus guide and to view their courses and time table for the semester.
